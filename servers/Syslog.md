#  Syslog Server

##  Overview
Central log collection server for firewalls, routers, and switches.

##  Purpose
- Collect logs from FortiGate  
- Store authentication logs (ISE + AD)  
- Monitor network events  
- Aid in troubleshooting  

##  Sources Logged
- FortiGate Traffic Logs  
- System Events  
- VPN events  
- Admin logins  

##  Testing
- Logs received from HQ FortiGate  
- Branch logs forwarded via VPN
