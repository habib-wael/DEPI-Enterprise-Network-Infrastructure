#  Certificate Authority (CA)

##  Overview
The Certificate Authority issues and manages digital certificates to secure communication across the enterprise.

##  Purpose in the Project
- Secure HTTPS for IIS  
- Support certificate-based authentication  
- Provide certificates to ISE  
- Enable encrypted communication for servers and firewalls  

##  Certificates Issued
- ISE Server Certificate  
- Web Server Certificate  
- RADIUS/TACACS certificates (optional)  

##  Testing
- HTTPS works with trusted certificate  
- ISE shows valid certificate chain
