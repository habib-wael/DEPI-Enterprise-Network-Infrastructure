#  Cisco ISE Integration with FortiGate

##  Overview
FortiGate integrates with ISE to receive user identity information for firewall policies.

##  Purpose
- Identity-based firewall rules  
- User-to-IP mapping  
- Enhanced access control  

##  Integration Steps
- Add FortiGate as Network Device in ISE  
- Configure RADIUS Shared Secret  
- Sync AD groups  
- Enable FSSO / RADIUS Accounting  

##  Testing
- Firewall detects username automatically  
- Policies applied based on user group
