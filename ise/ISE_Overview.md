##  What is Cisco ISE?
Identity Services Engine (ISE) is a centralized AAA and identity-based network access control platform.

##  Purpose in the Project
- Authenticate users & devices  
- Manage admin logins (TACACS+)  
- Apply identity-based firewall policies  
- Integrate with AD  

##  Features Used
- RADIUS (802.1X)  
- TACACS+  
- Profiling  
- Device Administration
