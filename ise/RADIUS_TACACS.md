#  RADIUS & TACACS+ Authentication

##  Overview
RADIUS handles network access authentication (802.1X), while TACACS+ handles device administration.

##  Purpose
- Control switch/router admin access  
- Enforce centralized authentication  
- Integrate with AD groups  
- Track admin logs  

##  Config Summary
- Added AD groups inside ISE  
- Defined Network Devices (Switches, Firewall)  
- Created RADIUS Policies  
- Created TACACS Command Sets  

##  Tests
- Verified RADIUS login through AD  
- TACACS admin login successful
